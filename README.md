# 美食地圖網站
羅列各項美食餐廳資訊

## 功能

1. 美食餐廳列表，包含餐廳名稱、類別、評分分數
2. 點擊圖樣可以查看詳細內容
3. 搜尋功能可以透過“名稱” “類別” 搜尋


## 套件/規格

- Node.js
- Express
- Bootstrap(v.5)
- Popper.js
- Jquery.js(3.6.0.min)

## 安裝

1.由github clone

```
git clone https://github.com/teddy821229/alpha_restaruant.git
```

2.安裝套件

```
npm install
```

3.啟動

```
npm run dev
```

4.如出現以下字樣，表示成功

```
Express is listening on localhost: 3000
```

5.以瀏覽器瀏覽 localhost:3000
